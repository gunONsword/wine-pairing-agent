# Wine Pairing — Vegetables & Vegetarian

Vegetables are the **most challenging category** for wine. Some — asparagus, artichoke, raw spinach, vinaigrette — actively fight wine. Others are surprisingly flexible. This skill covers both.

Read `wine-pairing-fundamentals` first.

---

## 1. The general framework

Vegetable + wine challenges:

- **Acid** (tomato, vinegar) → high-acid wine is required
- **Bitter** (artichoke, kale, endive, raw spinach) → wines taste duller / metallic
- **Sulphur** (asparagus, cabbage, Brussels sprouts) → grassy whites work, oak doesn't
- **Earth / umami** (mushroom, truffle, beet) → red wines surprisingly often work
- **Spice / herb** (basil, mint, thyme, garlic, ginger) → aromatic whites
- **Cooking fat** (olive oil, butter, cream) → varies

Default whites: **Sauvignon Blanc**, **Vermentino**, **Pinot Grigio**, **Grüner Veltliner**, **Verdejo**.
Default reds for veg: **Pinot Noir**, **Gamay**, **Frappato**, **Mencía**.

## 2. Salads

### 2.1 Caesar salad / green salad with vinaigrette

- The vinegar is the main challenge.
- **Concrete**: **Sauvignon Blanc** (NZ or Loire), **Verdejo**, **Albariño**, **Pinot Grigio**.
- Cut back vinaigrette acidity or replace lemon for vinegar to make any wine pairing possible.

### 2.2 Caprese (mozzarella + tomato + basil)

- **Sauvignon Blanc** (bodeboca canonical), **Vermentino**, **Pinot Grigio**, **rosé Provence**.

### 2.3 Greek salad (feta, olives, tomato, cucumber)

- **Assyrtiko** (Santorini canonical — regional)
- **Roditis**, **Vermentino**, **Sauvignon Blanc**.

### 2.4 Niçoise (tuna, egg, tomato, olive, anchovy)

- **Provence rosé** (Bandol, Côtes de Provence) — canonical Mediterranean regional
- **Tavel rosé**, **Sauvignon Blanc**.

### 2.5 Salade aux gésiers (duck gizzards salad — Sud-Ouest)

- **Bergerac rouge**, **Cahors** (Malbec), **Madiran**.

### 2.6 Salade de chèvre chaud (warm goat cheese on toast)

- **Sancerre / Pouilly-Fumé** (canonical Loire — same terroir as the cheese).

### 2.7 Cobb salad (American)

- **Pinot Noir**, **rosé**, **Sauvignon Blanc**.

### 2.8 Crab / lobster salad

- **Champagne**, **Sauvignon Blanc**, **dry Riesling**, **Albariño**.

### 2.9 Salade composée vinaigrée (mixed salad with vinaigrette)

- Use **rosé** or **young white** (papillesetpupilles — wine jeune leger).
- Or pair with a **light red** like **Beaujolais** if charcuterie is included (chamvermeil canonical: composed salad + light Beaujolais).

### 2.10 Salade d'été légères (cold pasta salad, rice salad, vegetable medleys)

- **Rosé** (vif et fruité), **Pinot Grigio**, **Vermentino**, **light Verdejo**.

## 5. Asparagus

The "wine killer" reputation comes from sulphur compounds. The right whites work brilliantly.

- **Concrete**:
  - **Sauvignon Blanc** (canonical — NZ, Loire, Touraine)
  - **Grüner Veltliner** (the Austrian go-to)
  - **dry Viognier** (rich asparagus dishes — weingraf)
  - **dry Riesling**
  - **Albariño**, **Verdejo**
  - **semi-dry White Burgundy** (with rich preparations — weingraf)
- Avoid: oaked Chardonnay, anything tannic.
- White asparagus with hollandaise: **Pinot Blanc / Pinot Gris Alsace**.
- Green asparagus with parmesan: **Grüner Veltliner**, **Sauvignon Blanc**.

---

## 6. Artichoke

Cynarin makes wines taste sweet. The trick: aromatic, slightly off-dry whites.

- **Concrete**:
  - **Vermentino di Sardegna** (Sardinian regional — artichokes love Vermentino)
  - **Verdicchio**, **Falanghina**
  - **Sauvignon Blanc**
  - **Bandol blanc**
  - **Grillo** (Sicilian)
  - **Pinot Bianco**

- Roasted asparagus with aceto balsamico: **Chianti Classico** (wineenthusiast).

- Crispy artichokes: **Soave** (wineenthusiast).

- Avoid: red wines, oaked whites.

## 14. Vegetable curries / Asian / soufflés / Mexican

- **Vegetable curry (Indian, Thai)** → off-dry Riesling, Gewürztraminer, Vouvray demi-sec
- **Wok / stir-fry with Asian greens** → dry Riesling, Pinot Gris Alsace, dry rosé
- **Vegetable terrine / quiche** → Sauvignon Blanc, Vouvray sec
- **Spinach soufflé** → White Burgundy, Champagne
- **Guacamole** → Sauvignon Blanc, Albariño, dry Riesling
- **Salsa verde / tomatillo** → Sauvignon Blanc, Verdejo
- **Bean burritos / vegan tacos** → Tempranillo, Garnacha, rosé
- **Chiles rellenos** → Albariño, Verdejo, Garnacha, Mencía

## 15. The "wine killers" — what to actually do

Foods that fight wine and how to rescue them:

| Killer ingredient | Why it fails | Wine that survives |
|---|---|---|
| Asparagus | Sulphur | Sauvignon Blanc, Grüner Veltliner |
| Artichoke | Cynarin sweetens wine | Vermentino, Verdicchio, off-dry whites |
| Vinaigrette (vinegar-heavy) | Acid spike | Drop vinegar, use lemon → high-acid Sauvignon Blanc |
| Garlic (raw, dominant) | Aggressive | Provence rosé, light Pinot Grigio |
| Spinach (raw or wilted) | Iron + bitter | Pinot Grigio, Sauvignon Blanc |
| Endive / chicory / arugula | Bitter | Soave, Provence rosé |
| Citron / lemon-heavy | Acid | High-acid Sauvignon Blanc, Albariño |
| Curry / chilli | Heat | Off-dry Riesling, Gewürz |
| Egg (boiled, deviled, mayonnaise) | Sulphur | Champagne, Sauvignon Blanc |

## 16. Abstract attribute mapping

| Vegetable trait | Wine attribute | Examples |
|---|---|---|
| Earthy / umami (mushroom, beet, truffle) | Pinot Noir, Nebbiolo | Burgundy, Bierzo, Alto Adige |
| Sweet caramelised (carrot, onion confit) | Off-dry, ripe | Riesling Spätlese, Viognier |
| Acidic (tomato, vinegar) | High-acid white | Sauvignon Blanc, Albariño, Cava |
| Bitter (kale, endive, artichoke) | Soft / off-dry / herbaceous | Vermentino, off-dry Riesling |
| Smoky (grilled vegetables, aubergine) | Pinot Noir or rosé | Bourgogne, Provence rosé |
| Cream / butter cooking | Chardonnay or crisp white | Mâcon, Soave |
| Herbal (basil, parsley, mint) | Aromatic herbal white | Sauvignon Blanc, Vermentino |
| Spicy (chilli, ginger) | Off-dry + low alcohol | Riesling Spätlese, Gewürz |

## 17. Quick decision tree

```
Vegetable-dominant dish?
├── Mostly tomato → Sangiovese / Albariño / Provence rosé
├── Vinaigrette/salad → Sauvignon Blanc / Verdejo
├── Asparagus → Sauvignon Blanc / Grüner Veltliner
├── Artichoke → Vermentino / Verdicchio
├── Mushroom → Pinot Noir / Nebbiolo / Mencía
├── Eggplant (Mediterranean) → Sangiovese / Aglianico / Nero d'Avola
├── Pumpkin/squash → White Burgundy / Riesling Spätlese
├── Beans/legumes (Iberian) → Tempranillo / Rioja Crianza / Bordón
├── Cassoulet → Madiran / Cahors / Saint-Mont
├── Choucroute (Alsace) → Riesling Trocken / Gewürz
├── Mediterranean ratatouille → Provence rosé / Bandol
├── Mexican (vegan tacos) → Tempranillo / Garnacha / Rosé
├── Asian veg curry/stir-fry → Off-dry Riesling / Gewürz
└── Vegan burger → Pinot Noir / Tempranillo
```

## 18. Common mistakes

- Big tannic red with vegetable-led plate → tannins amplify any bitterness
- Oaked Chardonnay with raw salad → oak overwhelms the freshness
- Dry sparkling with artichoke → cynarin makes the wine taste oddly sweet
- Red wine with asparagus → green flavours clash with tannin
- Heavy white with cold tomato gazpacho → mismatched textures; use Manzanilla
