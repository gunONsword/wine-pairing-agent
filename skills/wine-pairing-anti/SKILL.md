# Wine Pairing — Anti-Pairings (What NOT to Do)

The mirror of `wine-pairing-fundamentals`. Every reliable pairing principle has an inverse that produces a guaranteed mismatch. This skill catalogues those failures, the chemistry behind them, and the safer alternatives.

Use this when you want the agent to *warn* the user before suggesting a pairing, or to explain why a pairing the user proposed will fail.

---

## 1. The chemistry of mismatches

Five mechanisms account for almost every classic anti-pairing.

### 1.1 Iron + unsaturated seafood fat → fishy aftertaste

Red wine contains roughly **5–8× more iron** than white wine. When ferrous iron meets the unsaturated fatty acids in fish/shellfish, it triggers lipid oxidation that produces strong off-flavours — the so-called "fishy aftertaste".
*(Source: Tamura et al., *J. Agric. Food Chem.* 2009)*

Result: **most red wine + delicate seafood = unpleasant metallic / fishy mouthfeel.**

### 1.2 Tannin + protein in raw fish → astringent + bitter

Tannins bind to salivary proline-rich proteins (the lubricating layer). Without that lubrication, the mouth feels dry and astringent. With raw or lean fish there is no fat to soften the binding.

Result: **high-tannin red + sushi / sashimi / crudo = dry-mouth + amplified fish funk.**

### 1.3 Capsaicin + ethanol → heat amplification

Capsaicin binds to TRPV1 pain receptors. Ethanol both **dissolves capsaicin into the saliva** (distributing it more evenly across the tongue) and **directly stimulates TRPV1** itself. The two effects stack.

Result: **wine ≥14 % alcohol + chilli-spicy food = burning intensifies.** Common with oaky Chardonnay or Zinfandel and Thai / Sichuan / Vindaloo.

### 1.4 Cynarin → sweetness distortion

Artichokes contain cynarin (and chlorogenic acid). Cynarin blocks sweet receptors temporarily; after eating artichoke, water tastes sweet — and wine tastes oddly sweet too, which clashes with dry styles.

Result: **dry white or red + artichoke = wine reads oddly sweet, often metallic.**

### 1.5 Sulfur compounds → metallic / vegetal off-flavours

Asparagus, Brussels sprouts, cabbage, broccoli, kale and eggs contain sulfur compounds (methanethiol, dimethyl sulfide, glucosinolates) that mimic *common wine faults*. Hard-boiled or scrambled eggs also coat the mouth.

Result: **most wines + raw asparagus or cruciferous greens or plain egg = grassy / cabbagey / metallic.**

## 2. Catastrophic combinations (avoid absolutely)

| Combination | Why it fails | Better alternative |
|---|---|---|
| Tannic Cabernet Sauvignon + sushi/sashimi | Iron + tannin → metallic fishy aftertaste | Champagne BdB, dry Riesling, Sancerre |
| Syrah / Petit Verdot + oysters / lean white fish | Same iron + tannin clash | Chablis, Muscadet, Albariño |
| Big oaked Chardonnay + Thai / Indian spicy curry | High alcohol amplifies heat; oak clashes | Off-dry Riesling, Gewürztraminer |
| High-alcohol Zinfandel / Châteauneuf + chilli food | Ethanol-capsaicin synergy | Riesling Spätlese, Demi-sec Vouvray |
| Brut / Extra-Brut sparkling + sweet dessert | Wine reads as harsh / sour against the sugar | Moscato d'Asti, Demi-sec Champagne, Sauternes |
| Dry red + bittersweet chocolate cake | Both bitter → compound bitterness; dessert sugar strips wine | Banyuls, Maury, Vintage Port, Pedro Ximénez |
| Tannic Bordeaux + lamb with **mint sauce / vinaigrette** | Vinegar acid kills wine | Use lemon-juice mint sauce, then Bordeaux works |
| Big Cabernet + caviar | Tannin + brine = unpleasant metallic | Brut Blanc de Blancs, premium dry Riesling |
| Same Chianti for whole meal incl. dessert | Tomato-acid Chianti vs sweet finish → wine tastes thin | Chianti for primo; Vin Santo or Moscato for dolce |
| Rosé as "fits everything" with red meat + aged cheese | Lacks tannin / body for red meat; lacks sweetness for aged cheese | Use proper red for meat, sweet wine for blue cheese |
| Sweet vermouth / sweet aperitif before the meal | Saturates palate; ruins subsequent wines | Brut Champagne, Manzanilla, Fino |
| Buttery oaked Chardonnay + lobster *with drawn butter* | Too much fat on fat; cloying | Champagne BdB (contrast) or unoaked Chardonnay |
| Cabernet Sauvignon + Mexican shrimp ceviche | Tannin destroys the dish's citrus acidity | High-acid Sauvignon Blanc, Albariño, Manzanilla |
| Robust California Zinfandel + light tuna salad | Wine overwhelms lean fish | Provence rosé, Vermentino |
| Bold red (Petit Verdot, Syrah) + sole / flounder | Tannin amplifies fish funk | Chablis, Sancerre, Verdicchio |

## 3. "Wine killer" foods (and what to actually serve)

Foods that are notoriously hard to match. The wine you'd usually choose will be flattened, distorted, or clashed.

### 3.1 Artichoke (and cardoon)

- **Why it fails**: cynarin makes any wine taste oddly sweet; metallic edge.
- **Wines killed**: most dry whites and reds, especially oaked styles.
- **Rescue**: Sauvignon Blanc (high-acid, herbaceous), Vermentino di Sardegna, Fino sherry, Txakoli, off-dry Riesling.

### 3.2 Asparagus

- **Why it fails**: sulfur compounds (especially in green asparagus); methanethiol mimics wine faults.
- **Wines killed**: oaked Chardonnay, tannic reds, sweet wines.
- **Rescue**: Sauvignon Blanc (NZ Marlborough is the canonical fix), Grüner Veltliner, dry Viognier (rich asparagus dishes), Picpoul.

### 3.3 Vinegar / vinaigrette

- **Why it fails**: vinegar acidity overpowers wine acidity → wine tastes flat, washed-out.
- **Wines killed**: virtually all wines except the most acidic.
- **Rescue**: substitute lemon juice or wine reduction for vinegar; or use only very high-acid wine (Sauvignon Blanc, Albariño, Champagne).

### 3.4 Eggs (plain hard-boiled, scrambled, deviled)

- **Why it fails**: sulfur + mouth-coating yolk dulls everything; iron in yolk + wine tannin clash.
- **Wines killed**: most still wines.
- **Rescue**: Champagne / sparkling (acid + bubbles cut through coating); rich oaked Chardonnay if the egg is in a sauce (e.g. carbonara), where fat & dairy mediate; off-dry Riesling for deviled.

### 3.5 Cruciferous vegetables (Brussels sprouts, cabbage, broccoli, cauliflower, kale)

- **Why it fails**: sulfur compounds (glucosinolates → isothiocyanates).
- **Wines killed**: anything fruit-forward or sweet.
- **Rescue**: high-acid mineral whites (Sauvignon Blanc, Riesling Trocken), or pair with bacon/cream that masks the sulfur.

### 3.6 Spinach (raw or wilted), raw kale, endive, arugula

- **Why it fails**: bitter compounds + iron content.
- **Wines killed**: tannic reds (compounds bitterness), light unoaked whites (overwhelmed).
- **Rescue**: Soave Classico, Provence rosé, off-dry Riesling.

### 3.7 Tomato (raw or fresh sauce)

- **Why it fails**: high acid + slight sweetness.
- **Wines killed**: low-acid Pinot Grigio, oaked Chardonnay (oak clashes), big tannic reds (acid amplifies bitterness).
- **Rescue**: high-acid reds (Chianti, Sangiovese, Barbera), Provence rosé.

### 3.8 Pickles, kimchi, sauerkraut (uncooked)

- **Why it fails**: high acid + salt + vinegar.
- **Wines killed**: most dry wines.
- **Rescue**: Off-dry Riesling, Gewürz, Cava demi-sec, or a beer instead.

### 3.9 Mint sauce, fresh herbal-vinegar sauces

- **Why it fails**: vinegar + herb's volatile compounds collide with wine aromatics.
- **Wines killed**: even a fine Bordeaux next to vinegared mint sauce becomes a thin shadow.
- **Rescue**: switch to oil + lemon herb dressings; Bordeaux works.

### 3.10 Blue cheese (paired with dry red)

- **Why it fails**: high salt + funky alkan-2-ones + fat strip tannic dry reds.
- **Wines killed**: dry Cabernet, dry Syrah, dry Tempranillo.
- **Rescue**: **Sweet wines win**: Sauternes (Roquefort), Vintage Port (Stilton), PX (Cabrales), Tokaji Aszú, Banyuls, Vino Passito.

### 3.11 Chocolate (dry red wine + dark chocolate)

- **Why it fails**: chocolate has its own tannins + sugar; sugar strips wine's fruit, tannins compound, wine reads bitter.
- **Wines killed**: dry Cabernet Sauvignon, dry Nebbiolo, dry Bordeaux.
- **Rescue**: Banyuls (canonical), Maury, Rasteau, Vintage Port, PX, late-harvest Zinfandel, Recioto della Valpolicella. Wine must be **sweeter than the chocolate**.

### 3.12 Chilli / capsaicin-heavy food

- **Why it fails**: alcohol amplifies heat (capsaicin + ethanol synergy on TRPV1).
- **Wines killed**: any wine ≥14 % alcohol; oaked Chardonnay; tannic reds.
- **Rescue**: Off-dry Riesling Spätlese, Gewürztraminer, Demi-sec Vouvray, Moscato d'Asti for extreme heat.

### 3.13 Very salty cured / smoked foods (without sweetness)

- **Why it fails**: salt strips fruit from reds, makes oaky whites taste odd, makes high-alcohol bitter.
- **Wines killed**: oaked Chardonnay, high-alcohol red.
- **Rescue**: Sherry (Manzanilla, Fino, Amontillado), Champagne, Sauternes (with cured ham).

### 3.14 Very sweet desserts with a dry wine

- **Why it fails**: dessert sugar > wine sugar → wine tastes flat / sour / bitter.
- **Wines killed**: dry sparkling, dry whites, dry reds.
- **Rescue**: a wine sweeter than the dessert (Sauternes, Moscato, Tokaji, PX, Eiswein).

## 4. Anti-pairings by wine style

### 4.1 Tannic full-bodied red (Cabernet, Nebbiolo, Syrah, Tannat, Sagrantino)

**Avoid with**:
- Raw fish (sashimi, ceviche, tartare) — iron + tannin
- Delicate white fish (sole, cod, bass) — overwhelms
- Sweet desserts — wine tastes harsh
- Spicy chilli food — alcohol amplifies heat
- Bitter greens (raw kale, endive, artichoke) — compound bitterness
- Egg-based dishes — sulfur + iron clash
- Asparagus / cruciferous — vegetal off-notes

### 4.2 Oaked Chardonnay

**Avoid with**:
- Sushi / sashimi — oak collides with raw fish, soy
- Citrus-heavy dishes (lemon chicken, tarte au citron) — oak fights citrus
- Cold tomato dishes (gazpacho) — oak vs fresh acid
- Spicy curry — oak + heat = bitter
- Asparagus, artichoke — sulfur / cynarin distortion
- Salads with vinaigrette — vinegar kills oak

### 4.3 Brut / Extra-Brut sparkling (Champagne, Cava Brut Nature, Crémant Brut)

**Avoid with**:
- Sweet desserts (cake, tart, mousse) — acid amplified, wine sour
- Very sweet fruit (ripe mango, dried fig) — same
- Chocolate cake — wine tastes acidic, harsh
- Heavy stew or braise — bubbles wrong texture for richness
- Rare grilled red meat — wine too light

### 4.4 Off-dry / sweet white (Riesling Spätlese, Gewürz, Vouvray demi-sec, Tokaji)

**Avoid with**:
- Dry oysters (the sweetness flattens brine) — better with Champagne
- Tannic-rich grilled meat (sweetness clashes) — better with dry red
- Acid-heavy raw salads with vinaigrette

### 4.5 Light fruity red (Pinot Noir, Gamay, Mencía, Dolcetto)

**Avoid with**:
- Heavy braised meat / stew — wine disappears
- Big grilled steak / aged cheese — overwhelmed
- BBQ with sweet glaze — wine too delicate

### 4.6 High-alcohol red (Zinfandel, Châteauneuf, Amarone, Primitivo, Shiraz ≥15 %)

**Avoid with**:
- Spicy chilli food — heat amplified
- Light salad / vegetable plate — wine too heavy
- Sushi / raw fish — same iron-tannin clash
- Sweet dessert — wine alcohol burn

### 4.7 Rosé

**Avoid with**:
- Heavy red meat (steak, ribs) — rosé lacks tannin
- Strong aged cheese (Roquefort, Stilton, aged Cheddar) — rosé lacks sweetness for blue, body for hard
- Heavy braises and stews — wine too light

### 4.8 Buttery / oaky Chardonnay + lobster with drawn butter

**Why this surprising entry**: too much butter on butter = cloying, no contrast left.
**Better**: Champagne (contrast against fat) or unoaked Chardonnay.

## 5. The myth busters (popular "rules" that fail)

### Myth: "Red wine with meat, white wine with fish."
- **Fails** for oily fish (salmon, tuna, swordfish in red sauce), poultry in cream sauce, chicken curry, white meat in fruit reduction.
- **Reality**: match weight + acidity, not colour.

### Myth: "Rosé is a halfway compromise — it works with everything."
- **Fails** with red meat, blue cheese, sweet desserts.
- **Reality**: rosé is its own category with specific dishes (Provence cuisine, niçoise, ratatouille, summer salads).

### Myth: "Sparkling pairs with everything."
- **Fails** against sweet desserts (brut), heavy stews, rare red meat.
- **Reality**: choose the *style* of sparkling: Brut for savoury, Demi-sec for dessert, Blanc de Noirs for poultry.

### Myth: "Brut Champagne is the dessert wine of choice."
- **Fails** completely. The wine reads sharp / sour against sugar.
- **Reality**: Use Demi-sec, Doux, or another sweet wine.

### Myth: "Red wine always with cheese."
- **Fails** with most cheeses (the rind / salt / fat fight tannin).
- **Reality**: white wine wins most cheese pairings. Sweet wines win blue.

### Myth: "Old or expensive wine is always 'safer' for any meal."
- **Fails** because old wines are often delicate and fragile; pairing with a heavy braise destroys both.
- **Reality**: match the wine's *current* profile to the dish, regardless of age.

### Myth: "Same wine through the whole meal."
- **Fails** because dishes change in weight and flavour; one wine can't match aperitif + fish + meat + dessert.
- **Reality**: change wine across courses. Champagne aperitif → Sancerre with fish → Bordeaux with main → Sauternes with dessert.

### Myth: "Bubbles cut anything fatty."
- Mostly true but **fails** against pure heavy braise — the bubble texture is wrong, even if the acid works.
- **Reality**: bubbles + salty/fried = ✓; bubbles + heavy stew = mismatched texture.

### Myth: "Wine + chocolate = fancy."
- **Fails** unless wine is sweet and wine + chocolate match in intensity.
- **Reality**: pair only sweet wines or fortifieds with chocolate.

## 6. The "violated principle" anti-pairings

Each line is a fundamentals rule (from `wine-pairing-fundamentals`) plus its anti-pairing inverse.

| Fundamentals rule | Anti-pairing (what fails) |
|---|---|
| Wine acidity ≥ food acidity | Low-acid Merlot + tomato pasta → wine tastes flat |
| Wine sweeter than dessert | Brut sparkling + sweet cake → wine tastes harsh |
| Wine body ≈ dish body | Pinot Noir + braised short rib → wine disappears |
| Tannin binds protein + fat | Cabernet + sole → no fat to bind, tannin amplifies fish funk |
| Sweet wine for salty food | Dry red + Roquefort → tannin clashes with salt & funk |
| Off-dry / low-alcohol for spicy heat | Zinfandel + Vindaloo → burning intensifies |
| Match the sauce, not the protein | Pinot Noir + chicken in curry sauce → curry kills Pinot |
| Bitterness compounds, never cancels | Tannic Nebbiolo + raw artichoke → both bitter notes amplify |
| Sparkling cuts salty fried (but not sweet) | Brut Champagne + sweet tarte tatin → wine sharpens |
| Regional / terroir wins more often | Imported Cabernet + Tuscan pasta with anchovy → no concord |

## 7. The "what tastes wrong" diagnostic

If the user is already eating and the wine is failing, the *symptom* tells you the cause.

| Symptom | Probable mismatch | Fix next time |
|---|---|---|
| Wine tastes metallic | Iron + tannin (red + seafood/eggs) | Switch to white / sparkling |
| Wine tastes bitter / harsh | Sugar > wine sweetness OR oak collision | Sweeter wine OR unoaked |
| Wine tastes thin / flat | Food acid > wine acid | Higher-acid wine |
| Mouth feels burning | High alcohol + chilli | Lower alcohol, off-dry |
| Wine tastes "soapy" / "off" | Cynarin (artichoke) or sulfur (asparagus, eggs, brassicas) | Sauvignon Blanc, Riesling, or skip the wine |
| Wine tastes "watery" | Wine is overpowered by intense dish | Bigger / more structured wine |
| Mouth feels dried out | Tannin without enough fat | Switch to lower-tannin wine or richer dish |
| Wine tastes sour after dessert | Dessert sugar stripped wine | Use sweeter wine |
| Wine aroma vanishes | Wine is overchilled or food is too perfumed | Warm wine slightly; or simpler dish |

## 8. Cuisine-specific anti-pairings

### 8.1 Asian (Chinese, Thai, Indian, Japanese)
- ❌ Big oaked Chardonnay / Cabernet with curry → use off-dry Riesling
- ❌ High-alcohol red with sashimi → use Champagne / dry Riesling
- ❌ Brut sparkling with sweet teriyaki / unagi → use Demi-sec
- ❌ Tannic Nebbiolo with Sichuan numbing málà → use Gewürztraminer

### 8.2 Mexican / Latin
- ❌ Tannic red with mole poblano → bitter compound; use PX (sweet contrast)
- ❌ Bold Cabernet with ceviche → tannin destroys citrus; use Albariño / Manzanilla
- ❌ Heavy oaked Chardonnay with fish tacos → oak + lime collide; use Sauvignon Blanc

### 8.3 Mediterranean / Greek / Middle Eastern
- ❌ Big red with Greek salad → feta + vinaigrette kills the red; use Assyrtiko
- ❌ Brut Champagne with baklava → wine tastes harsh; use Mavrodaphne or Moscato
- ❌ Cabernet with hummus / yogurt mezze → tannin spoils; use rosé / Sauvignon Blanc

### 8.4 Italian
- ❌ Pinot Grigio with Bistecca alla fiorentina → too light; use Brunello
- ❌ Big Amarone with seafood pasta → too heavy; use Vermentino / Soave
- ❌ Brut Prosecco with Cantucci → use Vin Santo
- ❌ Tannic Barolo with eggplant Parmesan acidity → use Sangiovese

### 8.5 French
- ❌ Big Bordeaux + delicate Loire-style fish in cream → wine too heavy; use Chablis or Mâconnais
- ❌ Brut Champagne + Tarte Tatin → use Sauternes or Coteaux du Layon
- ❌ Cabernet Sauvignon + Munster cheese → use Gewürztraminer
- ❌ Sancerre + Boeuf Bourguignon → wine disappears; use Burgundy red

### 8.6 American
- ❌ Light Pinot Noir with BBQ ribs (heavy sweet sauce) → wine flattens; use Zinfandel
- ❌ Sauvignon Blanc with chocolate brownie → wine reads sharp; use Tawny Port / PX
- ❌ Cabernet with buffalo wings → tannin amplifies heat; use off-dry Riesling

## 9. Anti-pairings the agent should warn about

When the agent generates a recommendation, it should also flag these "near misses" where the user might naïvely pair badly:

1. If user has a **seafood/sushi** dish in mind and asks for "any red" → warn about iron/tannin clash, suggest Pinot Noir / Gamay if absolutely red is needed.
2. If user wants **sparkling with dessert** → warn unless they specify Demi-sec / Sweet.
3. If user wants **same wine through whole meal** → warn that palate will fatigue and the dessert match will fail.
4. If user proposes **red with chocolate** → warn unless wine is Banyuls / Maury / Port / PX.
5. If user proposes **oaked Chardonnay with citrus chicken / lemon sole** → warn about oak-citrus collision.
6. If user proposes **Cabernet with spicy curry** → warn about heat amplification.
7. If user proposes **big red with delicate fish (sole, bass)** → warn about overwhelm.

## 10. Quick "DO NOT" reference card

```
NEVER (or almost never):
  Brut sparkling + sweet dessert         (wine turns sour)
  Tannic red + raw fish / sushi          (metallic fishy taste)
  Tannic red + dark chocolate            (compound bitterness)
  High-alcohol wine + spicy chilli       (heat amplification)
  Dry red + Roquefort / Stilton          (use sweet instead)
  Oaked white + asparagus / artichoke    (sulfur / cynarin distortion)
  Old or fine wine + vinegar dressing    (vinegar kills wine)
  Big red + delicate white fish          (wine overwhelms)
  Same wine through whole meal           (palate fatigue + courses misaligned)
  Sweet aperitif before meal             (saturates palate)
```
