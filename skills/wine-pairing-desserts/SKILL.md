# Wine Pairing — Desserts

The cardinal rule: **the wine must be sweeter than the dessert**, otherwise the dessert strips the wine and it tastes harsh.

The most common mistake at dinner parties is serving Brut/Extra-Brut sparkling with sweet dessert — the result is always painful. Use Demi-sec, Doux, or sweet wine.

Read `wine-pairing-fundamentals` first.

---

## 1. The general framework

Dessert wine categories you should know:

| Category | Examples | Sweetness | Typical use |
|---|---|---|---|
| Late-harvest / passerillage | Sauternes, Coteaux du Layon, Vouvray moelleux | High | Foie gras, blue cheese, fruit tarts |
| Noble rot (botrytis) | Sauternes, Tokaji Aszú, Trockenbeerenauslese | Very high | Foie gras, blue cheese, crème brûlée |
| Ice wine | Eiswein (German), Icewine (Canadian) | Very high | Fruit tarts, mild blue cheese |
| Passito / dried-grape sweet | Vin Santo (Tuscany), Recioto della Valpolicella, Pantelleria | High | Cantucci, Tuscan biscuits |
| Fortified red | Vintage Port, LBV, Tawny | Variable | Stilton, chocolate, nuts |
| Fortified vin doux naturel | Banyuls, Maury, Rasteau | High | Chocolate (canonical) |
| Sherry sweet | Pedro Ximénez, Cream Sherry | Very high | Chocolate, cured ham, mousse |
| Demi-sec sparkling | Demi-sec Champagne, Asti Spumante, Moscato d'Asti | Variable | Fruity desserts, panettone |
| Sweet Riesling | Spätlese, Auslese, BA, TBA | Variable | Apple tarts, fruit-based desserts |
| Sweet Loire | Coteaux du Layon, Quarts de Chaume, Vouvray moelleux | High | Apple, pear, quince |

## 2. Fruit-based desserts

### 2.1 Apple tart / tarte tatin / apple crumble / strudel

- Caramelised apples sweetened butter — needs a wine that mirrors the sweetness with bright acidity.
- **Concrete**:
  - **Vouvray moelleux** / **Coteaux du Layon** / **Quarts de Chaume** (Chenin canonical with apple)
  - **Riesling Spätlese / Auslese**
  - **Eiswein**
  - **Sauternes** (especially with caramelised — apple tatin)
  - **Cider de glace** (ice cider, Normandy/Quebec)

### 2.2 Tarte aux fruits rouges / fresh berry tart / clafoutis aux cerises

- **Concrete**:
  - **Light Rosé** (chamvermeil — light fresh rosé with red fruit tart)
  - **Moscato d'Asti**
  - **Brachetto d'Acqui** (sweet sparkling red)
  - **Rosé Champagne demi-sec**
  - **Beaujolais Cru** (Morgon — for cherry clafoutis)

### 2.3 Tarte au citron / lemon tart / lemon meringue pie

- Acidic dessert demands sweeter wine.
- **Concrete**:
  - **Sauternes** (canonical — citrus mirror)
  - **Coteaux du Layon**
  - **Late-harvest Riesling**

### 2.4 Pear / quince tart / poached pears

- **Concrete**:
  - **Riesling Auslese**, **Vouvray moelleux**, **Champagne demi-sec**
  - **Pineau des Charentes** (with poached pears)

### 2.5 Apricot tart / abricot

- **Concrete**: **Sauternes** (apricot is a Sauternes flavour — strong concordance), **Riesling Spätlese**, **Coteaux du Layon**.

### 2.6 Strawberries / fraises au miel / strawberry shortcake

- **Concrete**:
  - **Bordeaux Clairet** (chamvermeil — strawberries with honey + Bordeaux Clairet)
  - **Moscato d'Asti**
  - **Rosé Champagne**
  - **Brachetto d'Acqui**

### 2.7 Pineapple / mango / passion fruit

- **Concrete**:
  - **Sauternes**, **Late-harvest Sémillon**, **Riesling BA**

### 2.8 Peach / nectarine

- **Concrete**: **Moscato d'Asti**, **Riesling Spätlese**, **Sauternes**.

### 2.9 Fruit salad (mixed)

- **Concrete**: **Moscato d'Asti**, **Asti Spumante**, **demi-sec Champagne**.

## 3. Chocolate

The trick: chocolate has tannin (bitter) and sugar. Match both with a wine that has tannin (concordance) AND sweetness (matches sugar).

### 3.1 Dark / bitter chocolate (≥70 %)

- **Concrete (canonical)**:
  - **Banyuls** (Roussillon vin doux naturel — Grenache-based, the canonical French chocolate pairing)
  - **Maury**, **Rasteau** (similar VDN)
  - **Vintage Port**, **LBV Port**
  - **Pedro Ximénez** (Andalucian sweet sherry — raisin, fig)
  - **Late-harvest Zinfandel** (American — for dark dessert chocolate)
  - **Recioto della Valpolicella** (Italian sweet red)
  - **Vino de Piedra** (Mexican Cabernet-based, see elixirs-dexception canonical pairing — round, evolved tannins match dark chocolate's character)

### 3.2 Milk chocolate / pralines

- **Concrete**: **Tawny Port** (10-20 yr), **Pedro Ximénez**, **Madeira Bual**, **Banyuls Rimage**.

### 3.3 White chocolate

- **Concrete**: **Sauternes**, **Coteaux du Layon**, **Vouvray moelleux**, **Asti Spumante**.

### 3.4 Chocolate fondant / lava cake / molten chocolate

- **Concrete**: **Banyuls**, **Maury**, **Vintage Port**, **PX Sherry**.

### 3.5 Chocolate mousse

- **Concrete**: **Banyuls**, **Tawny Port**, **Sauternes** (for white chocolate mousse), **Asti Spumante** (lighter mousse).

### 3.6 Brownie

- **Concrete**: **Tawny Port**, **PX Sherry**, **late-harvest Zinfandel**, **Niniches au chocolat + Bordeaux liquoreux** (chamvermeil pairing).

### 3.7 Chocolate truffles

- **Vintage Port**, **PX Sherry**.

### 3.8 Sachertorte / chocolate cake with apricot jam

- **Banyuls**, **Late-harvest Riesling**, **Sauternes**, **Tokaji Aszú**.

### 3.9 Tiramisu (coffee + mascarpone + cocoa)

- **Concrete**: **Vin Santo del Chianti**, **Marsala Dolce**, **Moscato d'Asti**, **Pedro Ximénez**, **Recioto**.

### 3.10 Chocolate + chilli / Mexican chocolate / mole-style desserts

- **PX Sherry**, **Banyuls**, **Recioto**, **late-harvest Zinfandel**.

### 3.11 Chocolate-licorice tart

- **Singlefile Malbec** (Margaret River — singlefilewines) or other Malbec / Carménère as a contrarian darker pairing for a powerful dessert.

## 13. Abstract attribute mapping

| Dessert trait | Wine attribute | Examples |
|---|---|---|
| Caramelised / brûlée | Botrytis or caramelised wine | Sauternes, Tokaji Aszú |
| Fresh fruit / berry | Light sparkling sweet | Moscato d'Asti, Brachetto |
| Citrus / lemon | Botrytis with citric finish | Sauternes |
| Chocolate bitter | Tannic + sweet fortified | Banyuls, Maury, vintage Port |
| Chocolate sweet | PX, Tawny Port | Pedro Ximénez, Tawny |
| Cream / custard | Sauternes or sparkling sweet | Sauternes, Moscato |
| Vanilla | Vouvray moelleux | Sauternes, late-harvest Riesling |
| Coffee / mocha | Marsala / Vin Santo / PX | Marsala dolce, Vin Santo, PX |
| Caramel / dulce de leche | PX / Madeira Bual | PX, Madeira Bual, Tawny Port |
| Nuts (pecan, almond, walnut) | Tawny Port / Madeira | Tawny Port, Madeira, Vin Santo |
| Spice (cinnamon, cardamom) | Aromatic late-harvest | Gewürztraminer VT, Tokaji |
| Honey | Botrytis or Madeira | Sauternes, Tokaji, Madeira |

## 14. Cardinal mistakes

❌ **Brut Champagne with sweet dessert.** The acidity of brut + dessert sugar = acid amplified = harsh, sour wine. Always Demi-sec or sweeter.

❌ **Sweet dessert wine with dry savoury cheese.** Mismatch (use semi-dry).

❌ **Red wine with fruit-based dessert.** Tomato / fruit acidity in dessert clashes with tannin.

❌ **Tannic dry red with chocolate.** The chocolate strips the wine. Use Banyuls, Maury, PX, or Port.

❌ **Sauternes with very fresh/light fruit (e.g. fresh strawberries plain).** Sauternes is too rich. Use Moscato or rosé.

❌ **Vin Jaune as dessert wine.** It's dry. Pair with the cheese course or savoury, not dessert.

## 15. Quick decision tree

```
Dessert type?
├── Fruit tart (apple, pear, apricot) → Sauternes / Vouvray moelleux / Riesling Auslese
├── Berry tart / fresh strawberry → Moscato d'Asti / Brachetto / rosé Champagne demi-sec
├── Lemon tart → Sauternes / late-harvest Riesling
├── Dark chocolate → Banyuls / Maury / Vintage Port / PX
├── Milk chocolate → Tawny Port / PX / Madeira Bual
├── Crème brûlée → Sauternes / Tokaji Aszú
├── Cheesecake → late-harvest Riesling / Sauternes
├── Tiramisu → Vin Santo / Marsala Dolce / PX
├── Cantucci → Vin Santo
├── Panettone → Moscato d'Asti / Recioto di Soave
├── Cannoli → Passito di Pantelleria / Marsala Dolce
├── Crème caramel / flan → Sauternes / PX
├── Ice cream (vanilla) → PX / Sauternes
├── Pumpkin/pecan pie → PX / Tawny Port / Madeira
├── Canelé bordelais → Sauternes / Bordeaux Crémant demi-sec
├── Apple strudel → Eiswein / BA / TBA

└── Turrón → Pedro Ximénez
```

## 16. Source notes

- French: chamvermeil (canelé + Sauternes; fraises miel + Bordeaux Clairet; niniches chocolat + Bordeaux liquoreux; banelés + Crémant Bordeaux demi-sec; tatin abricots + abricot-aromatique blanc; rosé léger + tartes fruits rouges), papillesetpupilles (charlotte bavaroise à la rhubarbe; tarte Bourdaloue; Kouign-Amann), elixirs-dexception (chocolat noir 75 %+ + Vino de Piedra rouge mexicain; sucré + vin moelleux/muté/liquoreux)
- Italian: enotecaravazzani (Moscato d'Asti + torta panna fragole concordance; vino passito + gorgonzola), dievole (cavallucci di Siena traditional + Vin Santo), perlagesuite (champagne + bisque gamberi for dessert-savory boundary)
- Spanish: bodegasamezola (turrón + maridaje trend with Crianza/Reserva), kommen (Eiswein + postres helados / tartas frutas; Sauternes + foie / tarta manzana caramelizada / queso azul; Tokaji + curry picante for contrast)
- English/Australian: singlefilewines (chocolate liquorice tart + Singlefile Malbec — bold Malbec for chocolate), wineenthusiast (chocolate + late-harvest Zinfandel, fruit-sauce-over-pork + rich Chardonnay)

For abstract principles see `wine-pairing-fundamentals`.
