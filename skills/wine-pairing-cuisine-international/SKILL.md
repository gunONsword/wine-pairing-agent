# Wine Pairing — International / World Cuisines

For each cuisine: standout dishes, default wine direction, and the spicy-vs-mild adjustment.

Read `wine-pairing-fundamentals` first.

---

## 1. Mexican cuisine

Wine should match the spice level and the sauce (moisture) of the dish.

### 1.1 Mild Mexican

- **Tostadas, quesadillas, mild fajitas** → Vouvray, Sauvignon Blanc, dry rosé, Albariño
- **Tacos de carnitas (pork)** → Tempranillo, off-dry Riesling
- **Chicken tostadas** → Vouvray (Chenin canonical — wineenthusiast)
- **Guacamole** → Sauvignon Blanc, Albariño
- **Salsa verde / fish tacos** → Sauvignon Blanc, Verdejo, Albariño
- **Enchiladas suizas (cream + cheese)** → Albariño, Mexican Chardonnay

### 1.2 Spicy Mexican

- **Mole poblano (chocolate + chilli + spices)** → Pedro Ximénez (contrast), Vino de Piedra (Mexican Cab — elixirs-dexception), late-harvest Zinfandel
- **Tacos al pastor (spit-roasted, pineapple)** → off-dry Riesling, Gewürztraminer
- **Enchiladas verdes / rojas** → Tempranillo, Mencía, Garnacha, off-dry Riesling
- **Pozole** → Tempranillo joven, Garnacha rosé
- **Chiles rellenos** → Mexican Cabernet, Garnacha
- **Chilaquiles** → Cava, Tempranillo joven, Vouvray

### 1.3 Mexican drinks tradition

- **Ceviche / aguachile** → Albariño, Sauvignon Blanc, Manzanilla
- **Churros con chocolate** → Pedro Ximénez
- **Tamales** → Mexican Tempranillo, Mencía

Mexican wines (Valle de Guadalupe — Mexican Cabernet, Nebbiolo, Tempranillo, Mexican Chardonnay) for traditional Mexican plates are the regional / terroir play — elixirs-dexception.

## 2. Chinese cuisine

### 2.1 Cantonese / Southern

- **Cantonese dim sum** → dry Riesling, Champagne
- **Roast duck (Cantonese style)** → Pinot Noir, off-dry Riesling
- **Yum cha brunch** → Champagne
- **Sweet and sour pork** → off-dry Riesling, Gewürz

### 2.2 Peking / Northern

- **Peking duck** → Pinot Noir (Oregon, Burgundy), off-dry Riesling
- **Dumplings (jiaozi)** → Champagne, dry Riesling, Pinot Gris

### 2.3 Sichuan / spicy

- **Mapo tofu / Sichuan spicy** → off-dry Riesling, Gewürz, Cava demi-sec
- **Kung Pao chicken** → off-dry Riesling, Gewürz
- **Dan Dan noodles** → Pinot Noir or off-dry Riesling
- **Hot pot (spicy Sichuan)** → Gewürz, off-dry Riesling

### 2.4 General Chinese

- **General Tso's chicken / sweet-soy sticky** → off-dry Riesling, Gewürztraminer
- **Lo mein / stir-fry noodles** → Riesling, Pinot Gris, Gewürz
- **Hong Shao Rou (red-braised pork)** → Pinot Noir, light Tempranillo
- **Char siu** → Pinot Noir, off-dry Riesling

## 3. Japanese cuisine

### 3.1 Raw

- **Sushi / sashimi** → Champagne Blanc de Blancs, dry Riesling, Albariño
- **Spicy tuna roll** → off-dry Riesling, Cava
- **Wasabi-heavy** → off-dry Riesling, Gewürz

### 3.2 Cooked

- **Tempura** → Champagne, Cava, dry Riesling
- **Tonkatsu** → Pinot Gris Alsace, Riesling, Grüner Veltliner
- **Yakitori** → Pinot Noir, Beaujolais Cru, light Riesling
- **Ramen (shoyu / shio)** → Pinot Noir, Riesling
- **Ramen (tonkotsu)** → Pinot Noir, Lambrusco
- **Sukiyaki** → Pinot Noir, light Sangiovese
- **Wagyu** → Cabernet Sauvignon, Bordeaux, Brunello, Champagne (a fascinating sommelier choice)
- **Unagi (sweet sauce eel)** → off-dry Riesling, Gewürz

### 3.3 Sweet

- **Mochi / wagashi desserts** → off-dry Riesling, light Moscato

## 7. Indian cuisine

### 7.1 Mild / cream-based

- **Korma (mild creamy)** → Viognier, off-dry Riesling
- **Butter chicken (murgh makhani)** → Viognier, off-dry Riesling
- **Pasanda** → Viognier, Marsanne
- **Saag paneer** → Riesling, Pinot Gris

### 7.2 Spicy

- **Tikka masala** → Gewürztraminer, off-dry Riesling
- **Madras / Vindaloo (spicy)** → Gewürztraminer, demi-sec Vouvray, off-dry Riesling
- **Lamb rogan josh** → Syrah, off-dry Riesling, Gewürz
- **Goan fish curry** → Gewürz, Riesling Spätlese

### 7.3 Tandoor / dry-cooked

- **Tandoori chicken** → off-dry Riesling, Spätburgunder
- **Seekh kebab** → Tempranillo, light Syrah
- **Naan / paratha** → off-dry Riesling

### 7.4 Vegetarian

- **Biryani** → off-dry Riesling, Gewürz
- **Dal / lentil curry** → off-dry Riesling, Albariño
- **Chana masala** → Gewürz, off-dry Riesling
- **Lassi-based / yogurt-based dishes** → off-dry Riesling

## 8. Middle Eastern / Mediterranean (non-European)

### 8.1 Lebanese / Levantine

- **Hummus, baba ganoush, tabouleh** → Provence rosé, Sauvignon Blanc
- **Kibbeh** → Lebanese Bekaa wine (Château Musar, Château Kefraya), Côtes du Rhône
- **Mezze platter** → Provence rosé, Cava, light Sauvignon Blanc
- **Shawarma** → Provence rosé, Tempranillo
- **Baklava** → Moscato d'Asti, Pedro Ximénez, Mavrodaphne

### 8.2 Turkish

- **Iskender kebab** → Tempranillo, Greek Agiorgitiko, Côtes du Rhône
- **Lahmacun (Turkish "pizza")** → Sangiovese, Tempranillo
- **Manti (Turkish dumplings)** → light Pinot Noir, Cava
- **Salmon tarator / tahini-lemon fish** → Riesling (singlefilewines)
- **Adana kebab** → Syrah, Cabernet, Tempranillo

### 8.3 Moroccan / North African

- **Tajine d'agneau au curcuma** → Saint-Joseph (Syrah, papillesetpupilles canonical), Provence rouge
- **Tajine de poulet aux citrons confits** → Côtes du Rhône blanc, Gewürz
- **Couscous royal** → Provence rouge léger, rouge fruité (elixirs-dexception)
- **Couscous + spicy harissa** → Tempranillo joven, rosé
- **Pastilla** → Riesling Spätlese, Vouvray demi-sec
- **Kefta brochettes** → Côtes du Rhône, structured rosé

### 8.4 Persian / Iranian

- **Kebab koobideh** → Tempranillo, Pinot Noir
- **Ghormeh sabzi** → Pinot Noir, Mencía
- **Tahdig** → off-dry rosé, Pinot Gris
- **Fesenjān (walnut-pomegranate stew)** → Pinot Noir, off-dry Riesling

## 9. American cuisine

### 9.1 BBQ / Southern

- **BBQ ribs / pulled pork / brisket** → Zinfandel, Shiraz, Malbec (canonical)
- **Fried chicken** → Champagne (canonical Kentucky style), Cava
- **Mac and cheese** → oaked Chardonnay, Lambrusco
- **Cornbread** → Riesling Spätlese, off-dry rosé
- **Gumbo** → off-dry Riesling, Pinot Noir, Beaujolais
- **Jambalaya** → Pinot Noir, Côtes du Rhône, off-dry Riesling
- **Crawfish boil** → Sauvignon Blanc, Champagne, Pilsner
- **Collard greens** → Pinot Noir, Beaujolais

### 9.2 Tex-Mex (American Mexican)

- **Chili con carne** → Zinfandel, Malbec
- **Burritos** → Tempranillo, Garnacha
- **Nachos** → Beaujolais, Tempranillo, Sauvignon Blanc

### 9.3 American mainstream

- **Hamburger / cheeseburger** → Zinfandel, Sangiovese, Malbec, Côtes du Rhône
- **Thanksgiving turkey** → Pinot Noir, Beaujolais Cru, Zinfandel, Champagne, Riesling
- **Buffalo wings** → off-dry Riesling, Zinfandel, Cava
- **Pumpkin pie / pecan pie** → Pedro Ximénez, Tawny Port, late-harvest Riesling
- **Cheesecake** → late-harvest Riesling, Sauternes

### 9.4 Cajun / Creole

- **Gumbo** → off-dry Riesling
- **Etouffée** → Riesling, dry Vouvray
- **Po' boys (fried)** → Champagne, Cava
- **Beignets** → Moscato d'Asti, Champagne demi-sec
- **Crab cake** → Sauvignon Blanc, Albariño

## 13. Common cross-cuisine mistakes

- High-tannin red with spicy Indian / Thai / Mexican → amplified heat. Pivot to off-dry Riesling.
- Champagne brut with sweet Asian desserts → use Moscato or demi-sec.
- Cabernet with raw sushi → tannin spoils fish; use Champagne / dry Riesling.
- Dry Riesling with sweet teriyaki → use off-dry (Spätlese).
- Tempranillo Gran Reserva with delicate fish curry → too big; use Albariño or off-dry Riesling.
- Big red with mole poblano → mole's bitter cocoa fights tannin. Use PX or late-harvest Zinfandel.

## 14. Quick spice-level pairing key

| Spice level | Default wine adjustment |
|---|---|
| None (mild Asian) | Dry Riesling, Pinot Gris, Champagne |
| Light spice (Vietnamese, mild Thai) | Riesling Halbtrocken, Gewürz, Pinot Gris Alsace |
| Medium spice (most Indian, Mexican enchiladas) | Off-dry Riesling Spätlese, Gewürztraminer |
| Hot (Vindaloo, Sichuan, mole) | Gewürz Vendange Tardive, Riesling Auslese, Moscato (last resort, very sweet) |
| Numbing (Sichuan málà) | Sparkling demi-sec or sweet Riesling |
