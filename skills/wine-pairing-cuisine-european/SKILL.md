# Wine Pairing — European Cuisines (by region)

The terroir rule first: regional cuisine + regional wine is rarely wrong. This skill maps Europe.

Read `wine-pairing-fundamentals` first.

---

## 1. France — by region

### 1.1 Bordeaux & Sud-Ouest

- **Foie gras** → Sauternes, Monbazillac, Jurançon moelleux
- **Joue de bœuf bordelaise** → Saint-Estèphe
- **Rillettes / pâté de canard** → Pauillac, Bordeaux Supérieur
- **Cuisses de poulet confites du Médoc** → Bordeaux Blanc (Château Malagar)
- **Seiches à la Ferret Capienne** → Graves blanc (Château Les Clauzots)
- **Cassoulet** → Madiran, Cahors, Côtes de Saint-Mont
- **Confit de canard** → Madiran, Cahors, Bergerac
- **Magret de canard** → Bordeaux rouge, Madiran, Cahors
- **Pommes sarladaises (duck fat)** → Cahors, Madiran, Bergerac rouge
- **Salade aux gésiers** → Bergerac rouge, Cahors, Madiran
- **Roquefort** → Sauternes (canonical)
- **Canelé bordelais** → Sauternes (canonical)
- **Fraises au miel** → Bordeaux Clairet

### 1.2 Burgundy / Beaujolais / Jura

- **Boeuf Bourguignon** → Burgundy Pinot Noir (Bourgogne, Pommard, Volnay)
- **Coq au vin** → Burgundy Pinot Noir
- **Œufs en meurette** → Bourgogne rouge
- **Jambon persillé** → Mâcon-Lugny, Burgundy white
- **Saint-Marcellin / Époisses** → Burgundy white or Marc de Bourgogne
- **Comté (24-36 mois)** → Vin Jaune, Château-Chalon
- **Beaufort** → Vin Jaune, Chasselas
- **Poulet aux morilles** → White Burgundy (Meursault, Puligny-Montrachet)
- **Beaujolais Cru region (volaille, lapin)** → Morgon, Fleurie, Moulin-à-Vent
- **Andouillette** → Beaujolais Cru, Chablis

### 1.3 Champagne

- Universal aperitif and oysters, smoked salmon, jambon de Reims
- **Champagne Brut** for the savoury course
- **Champagne Brut Rosé** for chicken liver pâté / cured meat
- **Champagne Vintage** for the fish main
- **Champagne Demi-sec** for dessert
- **Local cheese (Chaource, Langres)** → Champagne

### 1.4 Loire

- **Sancerre / Pouilly-Fumé** + Crottin de Chavignol / goat cheese
- **Saumur-Champigny / Chinon** (Cabernet Franc) + rabbit with prunes, terrine
- **Vouvray sec / demi-sec / moelleux** + apple tart, Brie, foie gras
- **Muscadet sur lie** + oysters / moules marinières
- **Quincy / Reuilly** + chèvre sec
- **Coteaux du Layon** + apple desserts, blue cheese (sweet contrast)

### 1.5 Alsace

- **Tarte flambée (flammekueche)** → Pinot Blanc, Riesling Trocken
- **Choucroute alsacienne** → Riesling Trocken, Gewürztraminer
- **Baeckeoffe (slow-cooked stew)** → Pinot Noir Alsace, Riesling
- **Munster cheese** → Gewürztraminer (canonical)
- **Kougelhopf** → Crémant d'Alsace, Tokay Pinot Gris VT

### 1.6 Provence / Côte d'Azur

- **Bouillabaisse** → Bandol rosé (canonical), Cassis blanc, Côtes de Provence
- **Aïoli** → Cassis blanc, Bandol blanc
- **Ratatouille** → Bandol rosé, Côtes de Provence rouge, Tavel rosé
- **Pissaladière** → Bandol rosé, Bellet
- **Pistou soup** → Provence rosé
- **Daube provençale** → Bandol rouge, Côtes de Provence rouge

### 1.7 Rhône valley

- **Côtes du Rhône reds** + game, daube, lamb tagine, ratatouille
- **Châteauneuf-du-Pape** + lamb, game, dark mushroom
- **Saint-Joseph (Syrah)** + duck, lamb, smoked
- **Condrieu (Viognier)** + lobster, white asparagus
- **Hermitage blanc** + foie gras, white fish in cream
- **Tavel rosé** + Niçoise salad, tuna

### 1.8 Languedoc-Roussillon

- **Cassoulet** → Côtes du Roussillon, Minervois
- **Banyuls** → chocolate desserts (canonical)
- **Maury** → chocolate, foie gras
- **Picpoul de Pinet** → oysters from Bouzigues, shellfish

### 1.9 Sud-Ouest specifics

- **Ossau-Iraty + black cherry jam** → Jurançon moelleux (regional)
- **Garbure** → Madiran, Cahors
- **Aligot** → Marcillac
- **Toulouse sausage / cassoulet** → Madiran, Cahors

### 1.10 Savoie / Mountain cuisine

- **Fondue savoyarde** → Apremont, Roussette de Savoie, Chasselas
- **Raclette** → Apremont, Vin de Savoie, Pinot Noir Alsace
- **Tartiflette** → Apremont, Roussette de Savoie

## 2. Italy — by region

### 2.1 Piedmont

- **Risotto al tartufo bianco** → Timorasso, Barolo, Barbaresco
- **Risotto allo zafferano alla milanese** → Lugana, aromatic rosato
- **Brasato al Barolo / al Barbera** → Barolo, Barbera d'Alba
- **Bollito misto** → Barbaresco, Nebbiolo d'Alba
- **Carne cruda all'albese (raw veal)** → Roero Arneis, Gavi
- **Tajarin al tartufo** → Barbaresco
- **Vitello tonnato** → Gavi (Cortese), Roero Arneis
- **Bagna cauda** → Arneis, Cortese, Erbaluce

### 2.2 Lombardy

- **Risotto alla milanese (with ossobuco)** → Barolo, Barbaresco, Rocca di Frassinello
- **Pizzoccheri** → Valtellina Superiore (Chiavennasca / Nebbiolo)
- **Cotoletta alla milanese** → Bonarda, Pinot Nero Oltrepò
- **Casoncelli alla bergamasca** → Valcalepio, Franciacorta

### 2.3 Veneto / Friuli / Alto Adige

- **Risi e bisi** → Soave Classico, Pinot Grigio Friuli
- **Bigoli in salsa (Venetian anchovy pasta)** → Soave, Custoza
- **Sarde in saor** → Soave Classico, Prosecco
- **Pasticcio di radicchio** → Pinot Nero Alto Adige, Lagrein
- **Polenta e schie (shrimp)** → Soave, Prosecco
- **Speck Alto Adige** → Lagrein, Schiava (Vernatsch)
- **Pasta with porcini** → Pinot Nero Alto Adige, Cabernet Franc Friuli

### 2.4 Emilia-Romagna

- **Tortellini in brodo** → Lambrusco di Sorbara
- **Lasagne alla Bolognese** → Sangiovese di Romagna Superiore, Lambrusco
- **Ragù alla bolognese (tagliatelle)** → Sangiovese di Romagna
- **Parmigiano-Reggiano + balsamico** → Lambrusco, vintage Champagne
- **Prosciutto di Parma / San Daniele** → Lambrusco, Trebbiano d'Abruzzo
- **Mortadella** → Lambrusco

### 2.5 Tuscany

- **Bistecca alla fiorentina** → Chianti Classico Riserva, Brunello di Montalcino
- **Pici al ragù di cinghiale** → Vino Nobile di Montepulciano, Chianti Classico Riserva
- **Pappa al pomodoro / Ribollita** → Chianti Classico, Sangiovese
- **Crostini toscani (chicken liver)** → Vin Santo del Chianti (traditional dipping), Chianti
- **Pecorino di Pienza** → Vino Nobile, Vermentino
- **Cantucci** → Vin Santo
- **Cavallucci di Siena (spice biscuits)** → Vin Santo

### 2.6 Marche / Abruzzo / Lazio

- **Spaghetti alle vongole** → Verdicchio dei Castelli di Jesi
- **Brodetto adriatic fish stew** → Verdicchio, Pecorino IGT
- **Olive ascolane** → Verdicchio
- **Cacio e pepe** → Frascati Superiore, Trebbiano d'Abruzzo
- **Carbonara** → Frascati Superiore, Cesanese del Piglio
- **Amatriciana** → Cesanese del Piglio, Montepulciano d'Abruzzo
- **Saltimbocca alla romana** → Frascati, Cesanese
- **Abbacchio (Roman lamb)** → Cesanese del Piglio, Montepulciano d'Abruzzo
- **Coda alla vaccinara** → Cesanese, Cannonau

### 2.7 Campania / Naples

- **Pizza Margherita / Marinara** → Aglianico (young), Falanghina (white alt)
- **Spaghetti alle vongole** → Greco di Tufo, Falanghina
- **Mozzarella di bufala caprese** → Falanghina, Greco di Tufo
- **Parmigiana di melanzane** → Aglianico, Taurasi (light)
- **Pesce all'acqua pazza** → Falanghina, Fiano di Avellino
- **Babà al rum** → Lacryma Christi del Vesuvio, Moscato

### 2.8 Sicily / Sardinia

- **Pasta alla Norma** → Nero d'Avola, Grillo
- **Spaghettone con polpo (octopus)** → Le Ginestre (Castellare Chardonnay-SB), Vermentino
- **Caponata** → Cerasuolo di Vittoria, Frappato, Nero d'Avola
- **Sicilian red prawns** → Etna Bianco, Vermentino, Champagne
- **Cannoli** → Passito di Pantelleria, Malvasia delle Lipari
- **Cassata siciliana** → Marsala Dolce, Passito di Pantelleria
- **Porceddu sardinian roast pig** → Cannonau di Sardegna
- **Bottarga (cured tuna roe)** → Vermentino di Gallura
- **Couscous trapanese (Trapani)** → Etna Bianco, Vermentino

## 3. Spain — by region

### 3.1 La Rioja

- **Caparrones con sus sacramentos** → Bordón Crianza (francoespanolas)
- **Chuletillas de cordero al sarmiento** → Rioja Gran Reserva (canonical)
- **Patatas a la riojana** → Tempranillo Crianza
- **Manchego** → Rioja Crianza (young), Gran Reserva (aged)

### 3.2 Andalucía

- **Gamba blanca de Huelva** → Manzanilla La Kika (bodeboca)
- **Jamón ibérico bellota** → Manzanilla, Fino, Cava Brut Nature
- **Gazpacho / salmorejo** → Manzanilla, Fino
- **Pescaíto frito / fried fish** → Manzanilla
- **Ceviche / aguachile** → Manzanilla (sherry.wine)
- **Cabrales (blue)** → Pedro Ximénez
- **Dorada con salsa de Amontillado** → Amontillado (sherry.wine)
- **Cordero / lamb (sweet preparations)** → Pedro Ximénez, Tempranillo Reserva

### 3.3 Galicia

- **Pulpo a la gallega** → Albariño, Godello
- **Empanada gallega** → Albariño, Godello
- **Hake / merluza in green sauce** → Albariño
- **Mariscada (mixed shellfish)** → Albariño, Godello
- **Cardo navideño (cardoon)** → Ribeiro (Martín Os Pasás)
- **Tarta de Santiago** → Moscatel, Pedro Ximénez

### 3.4 Basque Country

- **Pintxos / tapas** → Txakoli, Cava, Manzanilla
- **Bacalao al pil-pil** → Txakoli, Albariño
- **Chuletón de buey vasco** → Rioja Gran Reserva, Ribera del Duero Reserva
- **Idiazabal cheese** → Txakoli, Rioja Crianza
- **Marmitako (tuna stew)** → Txakoli, Rioja Rosé

### 3.5 Catalonia

- **Pa amb tomàquet** → Cava Brut
- **Calçots with romesco** → Cava Brut Nature
- **Suquet de peix** → Cava, Penedès Xarel·lo
- **Fideuá** → Cava, Penedès white
- **Crema catalana** → Pedro Ximénez, Moscatel
- **Botifarra negra (blood sausage)** → young Tempranillo

### 3.6 Central Spain (Madrid, Castilla)

- **Cocido madrileño** → Cava Brut / Brut Nature (estudiahosteleria)
- **Callos a la madrileña (tripe)** → Tempranillo Crianza
- **Lechazo / lechón asado (Castilian roast lamb / pig)** → Ribera del Duero Reserva, Toro
- **Mencía-based dishes (Bierzo)** → Mencía itself

### 3.7 Valencia / Murcia

- **Paella valenciana (chicken, rabbit, beans)** → Tempranillo joven, rosé Navarra
- **Paella de marisco** → Albariño, Verdejo, Cava
- **Arroz negro** → Cava Brut Nature, Albariño
- **Salt cod / bacalao** → Cava, Manzanilla
- **Turrón** → Pedro Ximénez (bodegasamezola trend)

### 3.8 Across Spain — pairings by tradition

- **Tapas variadas** → Manzanilla, Cava Brut Nature, Verdejo
- **Embutidos / charcuterie** → Crianza Tempranillo, Manzanilla
- **Quesos curados** → Rioja Reserva
- **Spicy chorizo / chorizo a la sidra** → Tempranillo Crianza

## 5. Portugal & British Isles & Other Europe

### 5.1 Portugal

- **Bacalhau à brás / à gomes de sá** → Vinho Verde, Alvarinho
- **Caldo verde** → Vinho Verde
- **Pastel de nata** → Moscatel de Setúbal, Late-bottled Vintage Port
- **Polvo à lagareiro** → Alvarinho, Douro white

### 5.2 British Isles

- **Fish & chips** → Champagne, English sparkling, Cava
- **Sunday roast (beef + Yorkshire)** → Bordeaux red, Pinot Noir
- **Stilton with Christmas crackers** → Vintage Port (canonical)
- **Sticky toffee pudding** → Tawny Port, Pedro Ximénez

### 5.3 Greece (also see Mediterranean)

- **Moussaka** → Agiorgitiko, Sangiovese, Côtes du Rhône
- **Souvlaki** → Agiorgitiko, Provence rosé
- **Greek salad** → Assyrtiko (canonical Santorini), Roditis
- **Spanakopita** → Assyrtiko
- **Baklava** → Mavrodaphne (canonical Greek sweet)
- **Kleftiko / lamb** → Agiorgitiko, Xinomavro

### 5.4 Scandinavia

- **Gravlax / smoked salmon** → Aquavit (traditional non-wine), Champagne, dry Riesling
- **Köttbullar (Swedish meatballs)** → Pinot Noir, light Tempranillo
- **Surströmming** → Aquavit, Vodka (not wine)

## 7. Common cross-regional mistakes within Europe

- Serving a young Bordeaux with delicate Loire-style fish → use a Loire white
- Big Italian Amarone with seafood → too heavy; switch to Soave or Vermentino
- Spanish Tempranillo Gran Reserva with fresh Galician fish → mismatch; use Albariño
- Brut sparkling with Tuscan dessert (cantucci) → use Vin Santo
- Crémant d'Alsace too cold with Munster → let it warm slightly so Gewürz-style off-dry profile emerges (or use Gewürz directly)
